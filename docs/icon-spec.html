Icon 图标 定义

统一动作语义表达与跨业务一致性图形系统。规范唯一来源为 JDSP 文件 1958051135088508929 的「1.3 icon图标 🟢」页，锚点节点 1487:10277。

行为准则

  • 统一语义优先：同一语义跨场景保持形态一致，仅调整尺寸与色值。
  • 15.0 主流程一致：视频详情、逛双列、评论区、搜索结果、短剧统一 48 母版网格。
  • 情感层级控制：高唤起动作强化反馈，中性动作降低噪声。
  • 单色继承策略：运行态统一使用 currentColor 驱动默认/激活/禁用颜色。

Icon 图标 结构

目录结构分为 line、fill、interactive、legacy 四类；互动区图标在 50×50 热区中以 36px 视觉盒渲染，并与计数区组成 50×62 单项容器。

错误示例

  • 引用非指定 JDSP 页面作为规范来源。
  • 线/面跨分区混用，或将 loose icon 节点直接投产。
  • 在资产内部写死业务态颜色，而非使用 currentColor。
  • 删除废弃图标而非迁移到 legacy 归档。
