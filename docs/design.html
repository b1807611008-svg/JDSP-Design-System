底部导航栏 定义

底部导航栏(Tabbar)位于屏幕底部,作为页面导航的最高层级,管控全局,点击切换整页内容。它分为常规底导和 Joy Agent 组合底导两种形态,支持 2~5 个坑位 + 可选灵动岛运营资源位,并在不同 OS 端/版本下选择不同材质(iOS 26+ 液态玻璃 / Android 与 iOS 老系统毛玻璃)。

行为准则

基础切换:点击坑位,切换整页内容(章节 01) 状态切换:默认态 ↔ 选中态(图标 + 文本变 jdred,背景变 gray_6) 灵动岛激活:对应坑位选中态背景消失;同时 Joy Agent(若存在)向外抽缩 16 → 28 DP

底部导航栏 类型

待补充。

底部导航栏 结构

Tabbar 在 Relay 中按「原子 → 组件」两层组织。Atom 是不可拆的视觉单位,组件层只做布局组合,不修改 atom 内部规则。理解层级有助于复用和约束。
