## 变体维度概览

| 维度 | 取值 | 说明 |
|---|---|---|
| 展示模式 | 左右滑动 / 全部 tab | 控制是否出现右侧加号入口和横向滚动区 |
| item 状态 | 选中态 / 非选中态 | 控制背景、文字颜色和字重 |
| 入口 | 有加号 / 无加号 | 左右滑动模式含加号，全部 tab 示例无加号 |
| 文案长度 | 2 字 / 4 字 | 示例覆盖「精选」「时尚穿搭」等长度 |

## 组合规则

1. 同一行只允许一个选中项。
2. 横滑模式下，滚动区和加号入口并列展示。
3. 全部 tab 模式下，当前稿件未展示加号入口。
4. 文字过长时 item 宽度可自适应，但不得压缩高度。
5. 当前样例中 item 宽度为 52 px，适合 2 字文案；4 字文案需要确认是否完整展示或截断。

## Donts

- 不要同时出现多个选中态。
- 不要用红色实底表达选中，当前规范为浅红底 + 红字。
- 不要让加号入口跟随横向列表滚动。
- 不要把二级 Tab 用作一级频道导航。
- 不要在 item 内放图标、角标或多行文字，当前结构仅支持单行文本。
