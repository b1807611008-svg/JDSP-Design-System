## 维度规则

### 媒体类型

| 取值 | 规则 |
|---|---|
| 多图 / 图文 | 可在封面左上角 8 px 处展示 13 x 13 `icon-copy-fill` 图标 |
| 视频 | 可在封面左上角 8 px 处展示 16 x 16 播放胶囊，播放图标来自 `assets/tag-video.svg` |
| 直播 | 可在封面左上角 8 px 处展示直播中 / live 标签，图标来自 `assets/tag-live-on.svg` / `assets/tag-live.svg` |
| 浏览提示 | 可展示刚刚看过、110万观看、回放等半透明胶囊 |
| 趋势提示 | 可展示飙升值红色强调胶囊，趋势图标来自 `assets/tag-rise.svg` |

### 场景

| 取值 | 规则 |
|---|---|
| 普通 | 白底卡片 + 封面 + 标题 + 作者 + 点赞或浏览量 |
| 种草个人页 | 白底卡片 + 话题渐变条 + 种草数 |
| 挂商卡 | 在内容区顶部插入商品挂载条，标题最多两行 |
| 沉浸式 | 封面覆盖整卡，不展示普通白底内容区 |

### 商卡类型

| 类型 | 规则 |
|---|---|
| B2C | 商品图 + 现价 + 原价 + 已售数量 |
| O2O | 门店图 + 门店名 + 距离 |
| 景点 | 景点图 + 景点名 + 距离 |
| 评分 | 商户图 + 商户名 + 分数 |

### 文本状态

| 状态 | 规则 |
|---|---|
| 一行文案 | 单行标题 |
| 双行文案 | 两行标题，行间距 6 pt |
| 单行营销标 | 14 px 高营销标 + 单行标题 |
| 双行营销标 | 首行营销标 + 标题，第二行标题；行间距 6 pt |

内容区高度由文本行数、商卡和用户信息区共同决定；一行文案不得强行撑成双行卡高度，底部内边距固定 8 px。

### 反馈类型

| 取值 | 图标 / 状态 | 说明 |
|---|---|---|
| 点赞 | `icon-heart.svg` + 数字 | 常规线性图标，支持未点赞 / 已点赞切换；激活态仅 icon 使用 `#ff0f23`，数字保持灰色，示例 `313` |
| 浏览量 | `icon-browse.svg` + 数字 | 默认使用 `#828794`，示例 `1.2w` |
| 种草 | `icon-like.svg` + 数字 | 常规线性图标，支持未种草 / 已种草切换；完成态仅 icon 使用 `#15ba64`，数字保持灰色，示例 `256` |
| 无 | 不展示 meta row | 沉浸式图文 / 视频当前样例不展示指标 |

### 标题行数

| 取值 | 规则 |
|---|---|
| 单行 | 13 px，超出截断 |
| 双行 | 挂商卡可展示两行标题，行高 13 px |
| 沉浸式标题 | 12 px，置于底部渐变区 |

## 组合优先级

1. 媒体封面必须展示。
2. 媒体类型标识优先于弹幕、话题、商品信息。
3. 商品挂载条优先展示价格、原价、销量，不承载更多商品卖点。
4. 标题优先保证可读性；挂商卡中商品条不得压缩标题到不可读。
5. 话题条只出现在种草个人页类变体中。
6. 沉浸式卡不展示普通 meta row，避免破坏封面完整性。

## Donts

- 不要在同一张卡里同时叠加弹幕、话题、商品条和多个角标。
- 不要把沉浸式卡强行加普通白底内容区。
- 不要让商品条超过一行核心交易信息。
- 不要在无视频内容时展示视频标签。
- 不要把点赞、浏览量与种草数混排在同一指标位。
