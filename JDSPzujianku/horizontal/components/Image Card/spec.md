# 基础内容卡 · 视觉规范

> design.md → [index](./design.md) · 同 bundle: [variants](./variants.md) · [behaviors](./behaviors.md)

## 预览

Relay 根节点 `1042:4844` 是 498 x 1612 的组件展示面板，包含 8 个内容卡 symbol。当前可见规格集中在 176 px 宽度，普通内容卡高度 289 px，挂商卡高度 321-340 px，沉浸式卡高度 234 px。

## 指标 Icon 映射

| 指标类型 | Icon.md 语义 | SVG 资产 | 颜色规则 |
|---|---|---|---|
| 点赞 | `icon-heart` | `src/assets/icons/icon-heart.svg` | 默认态 icon 与数字均为 `#828794`；激活态仅 icon 为 `#ff0f23`，数字仍为 `#828794` |

指标槽以 `icon + 数字` 呈现，三类指标 icon 均来自常规 Icon.md 线性图标库，显示尺寸均为 12 x 12 pt；点赞与种草支持默认态 / 激活态切换，激活态只改变 icon 颜色，数字始终保持辅助灰；不使用右侧互动区专属 `interactive/` icon；种草不使用状态文案；数值过长时按业务缩写为 `1.2w` / `1.2万`。
