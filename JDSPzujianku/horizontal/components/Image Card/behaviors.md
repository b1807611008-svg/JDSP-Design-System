# 基础内容卡 · 行为规范

> design.md → [index](./design.md) · 同 bundle: [spec](./spec.md) · [variants](./variants.md) · schema: [ai-schema.yaml](./ai-schema.yaml)

## 应用场景

| 场景 | 使用形态 | 说明 |
|---|---|---|
| 内容瀑布流 | 图文 / 视频 | 推荐、发现、种草流中的基础内容曝光 |
| 种草个人页 | 种草个人页专用 / 挂商卡 | 展示作者内容、话题和种草数 |
| 商品关联内容 | 挂商卡 / B2C 挂商卡 | 内容卡内轻量承接商品信息 |
| 沉浸式内容入口 | 沉浸式图文 / 视频 | 更强图片或视频封面表达，弱化作者和指标 |
| 视频氛围预览 | 视频卡带弹幕 | 用弹幕标签表达内容氛围 |

## 点击热区

| 热区 | 行为 | 优先级 | 说明 |
|---|---|---:|---|
| 整卡 | 进入内容详情或视频播放页 | 1 | 默认主行为 |
| 商品挂载条 | 进入商品详情或商详半屏 | 2 | 与整卡冲突时商品条优先 |
| 话题渐变条 | 进入话题页 / 聚合页 | 2 | 仅当业务支持话题点击时启用 |
| 右下角指标槽 | 点赞 / 浏览量 / 种草数展示或反馈 | 3 | 同一张卡只启用一种指标类型；点赞和种草支持默认态 / 激活态切换；使用常规 Icon.md 线性图标，不使用右侧互动区专属 icon |
| 弹幕标签 | 可点击或纯展示 | 4 | 默认纯展示；点击能力需业务明确 |
| 视频 / 图文标签 | 纯展示 | - | 不建议独立点击 |
| 左上角卡片标签 | 纯展示 | - | 位于封面左上 8 px，覆盖图文、视频、直播中、刚刚看过、观看量、飙升值、回放、live，不建议独立点击 |

## 状态反馈

| 状态 | 反馈 |
|---|---|
| 默认 | 卡片无额外状态色，点击反馈由页面容器提供 |
| 未点赞 / 未种草 | 指标 icon 与数值均使用 `#828794` 辅助灰 |
| 按下 | 建议整体 opacity 或背景轻微变化，避免封面位移 |
| 点赞成功 | 仅点赞 icon 使用 `#ff0f23`，数值保持 `#828794`，数值可保持不变或按接口更新 |
| 取消点赞 | 点赞 icon 回到默认态，数值可保持不变或按接口更新 |
| 种草成功 | 仅种草 icon 使用 `#15ba64`，数值保持 `#828794`，数值可保持不变或按接口更新 |
| 取消种草 | 种草 icon 回到默认态，数值可保持不变或按接口更新 |
| 商品条按下 | 商品条局部反馈，不影响整卡封面和标题 |
| 图片加载中 | 使用卡片圆角占位，不改变卡片尺寸 |
| 图片加载失败 | 使用默认占位图或灰底，不隐藏标题与作者 |

## 内容溢出

| 内容 | 规则 |
|---|---|
| 标题 | 单行卡片截断；挂商卡允许两行 |
| 内容区高度 | 随标题行数、商卡和底部信息区自适应；文案 / 信息区到底部距离始终保持 8 px |
| 作者昵称 | 单行截断，不能挤压右侧指标 |
| 点赞 / 浏览量 / 种草数 | 保持右侧最小宽度，过长时可缩写为 `1.2万` |
| 商品价格 | 现价优先；空间不足时隐藏原价，再隐藏销量 |
| 图片下方商卡 | 文本过长时主标题截断；右侧距离 / 分数 / 已售信息保持可读 |
| 弹幕标签 | 横向溢出裁切，不换行 |
| 话题文本 | 单行展示，过长截断 |

## 降级规则

| 缺失内容 | 降级 |
|---|---|
| 缺少封面 | 使用默认封面占位，仍保留标题和作者 |
| 缺少作者头像 | 隐藏头像或使用默认头像，昵称左对齐 |
| 缺少作者昵称 | 隐藏作者区，指标仍可展示 |
| 缺少指标 | 隐藏右侧指标，作者区占满宽度 |
| 缺少商品图 | 隐藏商品条，退化为普通种草卡 |
| 缺少商品价格 | 隐藏商品条，不展示半残商品信息 |
| 缺少左上角标签 | 隐藏标签，不影响封面和标题 |
| 缺少话题 | 隐藏话题渐变条 |
| 缺少弹幕 | 隐藏弹幕标签组，保留视频标签 |

## 动效建议

| 场景 | 动效 |
|---|---|
| 卡片按下 | 100-150ms opacity 反馈 |
| 点赞 / 种草成功 | icon 可做 160ms scale 反馈，数值 120ms fade update |
| 图片加载 | 占位到图片淡入 180ms |
| 商品条点击 | 商品条局部按压，不触发整卡二次反馈 |

## 禁止规则

- 不要让商品条和整卡同时触发两个跳转。
- 不要把视频 / 图文标签做成主按钮。
- 不要在商品信息不完整时展示半截商品条。
- 不要在沉浸式卡中补充作者和指标，除非设计稿新增对应变体。
- 不要为了容纳长标题压缩封面比例。
- 不要让弹幕标签换行，顶部只允许单行横向排列。
- 不要在右下角指标槽同时展示点赞、浏览量和种草数。
