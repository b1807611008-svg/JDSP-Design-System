# 双列 · 变体 Variants

> design.md → [index](./design.md) · 同 bundle: [spec](./spec.md) · [behaviors](./behaviors.md)

## 变体维度概览

- 内容卡高度：短卡 `176×231`、长卡 `176×328`
- 封面比例：`176×176`（1:1）与 `176×234`（约 3:4）
- 内容区：单行内容区 `176×37`、双行 + 营销信息内容区 `176×76`
- 标签：`卡片标签` 支持 16×16 图标型、75×16 文案型、13×13 小图标型
- 互动信息：浏览 `睁眼`、喜欢 `heart`
- 页面 tab：一级 tab 与二级 tab 均有吸顶示意

## 各维度详细规范

### 双列容器

| 属性 | 值 |
|---|---|
| 页面宽度 | 375 |
| 内容卡列宽 | 176 |
| 列间距 | 7 |
| 左右安全边距 | 8（由 375 - 176×2 - 7 推得） |
| 排列 | 左右两列瀑布流，卡片高度可不等 |

### 内容卡

| 形态 | 尺寸 | 组成 |
|---|---|---|
| 短卡 | 176×231 | 1:1 封面 + 单行标题 + UGC/互动区 |
| 长卡 | 176×328 | 3:4 封面 + 双行标题 + 营销信息 + 用户信息/互动区 |

### 封面

| 形态 | 尺寸 | 说明 |
|---|---|---|
| 1:1 | 176×176 | 商品/内容正方形图片 |
| 3:4 | 176×234 | 视频/直播/内容封面 |

### 内容区

| 形态 | 尺寸 | 说明 |
|---|---|---|
| 单行内容区 | 176×37 | 标题 1 行 + UGC/互动区 |
| 双行内容区 | 176×76 | 标题 2 行 + 营销信息 + 用户信息/互动区 |

### 互动区

- 用户头像：16×16
- 用户信息与互动区：160×16
- 互动信息：34×12
- 图标：12×12
