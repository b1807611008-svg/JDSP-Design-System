## 设计原则

- **不在本地仓库放位图**：所有切图通过 CDN URL 引用，仓库不存 PNG 二进制
- **每张切图都登记**：Relay 节点 ID + 用途 + 尺寸 + CDN URL + 状态
- **skill 侦测、人工上传**：skill 自动揪出 IMAGE fill 并登记 `⏳`，京东 CDN 上传由设计师 / 运维手动完成
