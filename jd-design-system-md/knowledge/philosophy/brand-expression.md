# ✨ 品牌表现

> 京东红 #fa2c19 是 20 多年沉淀的品牌资产。Joy 是 6 亿用户的情感连接。每一次大促是品牌资产的释放和再投资。
>
> 品牌不是装饰,是用户对"京东"两个字产生条件反射的视觉触发器。

---

## 1. 核心主张

| 主张 | 具体表现 |
|---|---|
| 京东红是品牌护城河 | 主 CTA / 价格 / 品牌区 严格使用 #fa2c19 |
| Joy 是情感锚点 | 在适合的场景出现(空状态 / 节庆 / 引导),不滥用 |
| 节庆基因要传承 | 618 / 双 11 / 春节 等大促有可识别的视觉资产体系 |
| 品牌克制即品牌强大 | 非品牌区不滥用红色,反而强化品牌区识别 |

---

## 2. 品牌色使用规则

| 区域 | 京东红使用 | 反例 |
|---|---|---|
| 主 CTA(加购 / 立购 / 提交)| ✅ 必须 | - |
| 价格 | ✅ 必须 | 价格用其他色 |
| 品牌区 / Logo / Tab Bar 选中态 | ✅ 必须 | - |
| 营销标识(限时 / 秒杀 / 优惠)| ✅ 推荐 | - |
| 次要操作 / 文字按钮 | ❌ 不允许 | 普通"查看更多"用红色 |
| 错误提示 | ❌ 改用语义红 `color.semantic.danger`(略偏) | 错误用品牌红 |
| 装饰图形 / 大块底色 | ❌ 不允许(品牌区除外) | 整个 Banner 红底 |

> **关键判断**:京东红的稀缺性 = 它的力量。如果一屏 30% 是京东红,那它就不再是品牌色了。

---

## 3. Joy 吉祥物使用边界

**Joy 应该出现的场景**:
- 空状态("购物车里啥都没有,Joy 邀你逛逛")
- 节庆引导(春节 / 618 / 双 11 主题视觉)
- 营销提醒(限时秒杀 / 大促预告)
- 客服 / 帮助中心(Joy 客服形象)

**Joy 不应该出现的场景**:
- PDP / 购物车 / 结算 / 支付主流程(打扰转化)
- 严肃信息页(订单详情 / 投诉处理 / 隐私政策)
- 错误反馈(对受挫用户用吉祥物显得轻浮)

**Joy 形象使用规范**:[[../../horizontal/brand/mascot-joy.md]]

## 4. 大促视觉体系

每个大促节点有独立的视觉资产包,但共享统一基因:

| 大促 | 主色基因 | 装饰风格 | 文档 |
|---|---|---|---|
| 618(年中) | 京东红 + 金 | 立体礼花 + 倒计时 | [[../../horizontal/brand/promotion/618.md]] |
| 双 11(年终) | 京东红 + 紫 | 梦幻渐变 + 烟花 | [[../../horizontal/brand/promotion/1111.md]] |
| 春节 | 京东红 + 金 | 民俗插画 + 福字 | [[../../horizontal/brand/promotion/seasonal.md#spring]] |
| 520 / 中秋 | 主题色因节而异 | 节日插画 | [[../../horizontal/brand/promotion/seasonal.md]] |

**统一基因**:京东红基底 + Joy 节庆形象 + 倒计时 / 优惠条 等共用元素

## 5. 子品牌的品牌边界

京东超市 / 京东健康 / 京东金融 / 京东物流 / 京东工业 等子品牌:

**必须保留**:
- 京东 Logo 出现位置(左上角 / 启动屏 / 我的页)
- 京东红作为关键 CTA / 价格色
- 京东品牌字体

**可以差异化**:
- 子品牌辅助色(京东健康加蓝 / 京东金融加金)
- 子品牌专属插画风格
- 子品牌专属吉祥物变体(如京东健康的 Joy 医生)

详见:[[../../horizontal/brand/sub-brands.md]]

## 6. 反面是什么

| ❌ 反面行为 | 解释 |
|---|---|
| 主 CTA 不用京东红 | 削弱品牌触发器 |
| 大块红色背景 | 稀释品牌色稀缺性 |
| Joy 滥用 | 破坏严肃信息的可信度 |
| 子品牌完全脱离京东视觉 | 用户认知断裂 |
| 大促视觉抢戏到识别商品困难 | 品牌表现 vs 聚焦实效失衡 |

---

## 7. 引用此原则的场景

- **新业务 Logo 设计**:是否符合京东品牌边界?
- **大促视觉评审**:是否传承了节庆基因?
- **Joy 形象使用**:场景是否合适?
- **子品牌差异化提案**:边界在哪?

---

## 8. 与其他原则的张力

**与"聚焦实效"**:大促首屏品牌氛围会牺牲屏效
**协调**:大促首屏(顶部 1-2 屏)品牌优先,主流程(PDP / Cart / Checkout)聚焦实效优先

**与"简洁统一"**:子品牌差异化会带来视觉多样
**协调**:简洁统一保 L1(品牌色 / 原子组件),子品牌差异在 L3(辅助色 / 插画风格)

---

## 9. 历史踩坑

| 时间 | 事件 | 教训 |
|---|---|---|
| 2024 双 11 | 全屏礼花动效 | 商品识别度下降 / 改为局部装饰 |
| 2025-Q2 | 某子品牌完全脱离京东红 | 用户调研发现"不像京东" / 加回品牌触点 |
| 2025-Q4 | Joy 出现在订单异常页 | 用户感觉被冒犯 / 移除 Joy |
