# 数据展示类组件 · Data

> 大量结构化数据的展示。**京东 APP 移动端不主用 Table,主用 List + Card**。

---

## 组件清单

| 组件 | 状态 | 用途 |
|---|---|---|
| **List** | stable | 列表(单列流 / 多列项) |
| **Table** | beta | 表格(iPad / 后台管理)|
| **Tree** | beta | 树形结构(分类层级) |
| **Calendar** | stable | 日历(预约 / 配送时间) |
| **Chart** | beta | 图表(订单统计 / 健康数据) |
| **DataGrid** | beta | 数据网格(高密度展示)|

---

## List 类型

| 类型 | 用途 | 行高 |
|---|---|---|
| 标准列表 | 订单 / 设置 | 56-72pt |
| 紧凑列表 | 消息 / 收藏 | 44-52pt |
| 卡片列表 | 商品 / 频道 | 自适应 |
| 双列流 | 商品流 | 自适应 |

---

## Table 移动端约束

移动端 Table 滥用是反例。优先级:
1. **能用 List 不用 Table**(列表更适合移动)
2. **必须用 Table 时**:横向滚动 + 首列固定
3. **iPad / 桌面端**:可使用完整 Table

---

## Calendar 组件

### 类型
| 类型 | 用途 |
|---|---|
| 单日选择 | 预约配送 |
| 区间选择 | 优惠券有效期 / 报告时段 |
| 多选 | 健康打卡 / 收藏日期 |

### 视觉
- 已选:京东红实心圆
- 区间:京东红连续区域(浅色 wash)
- 不可选:`text.tertiary`
- 今天:京东红描边圆

---

## Chart 类型

| 类型 | 用途 |
|---|---|
| 折线图 | 订单趋势 / 价格曲线 |
| 柱状图 | 销量对比 |
| 饼图 | 比例展示 |
| 散点图 | 高级数据 |
| 漏斗图 | 转化漏斗 |

**配色**:使用 `color.viz.*` Token,不直接用品牌色 / 语义色。

---

## 数据加载

- 首次加载:Skeleton(列表项形状还原)
- 加载下一页:底部 Spinner
- 无更多:文字"已经到底了"
- 错误:错误图标 + 重试按钮

---

## 反例

| ❌ 反面 | 解释 |
|---|---|
| 移动端 Table 强行展示 5+ 列 | 横向溢出 / 文字截断 |
| Chart 用品牌红做数据色 | 与品牌区混淆 |
| List 没有空状态 | 用户困惑 |
| 加载下一页无反馈 | 不知道有没有更多 |
