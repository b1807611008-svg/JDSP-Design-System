# 图像系统 · Imagery

> 商品图 / Banner / 插画 / 吉祥物 Joy 的视觉规范。**京东电商场景下,商品图是绝对主角**,其他图像类型为商品图让路。

---

## 1. 商品图

### 主图比例
| 场景 | 比例 | 备注 |
|---|---|---|
| PDP 主图 | 1:1 | 全平台默认 |
| 双列卡 | 1:1 或 4:5 | 取决于卡型 |
| 横版海报 | 16:9 / 21:9 | 频道页 / 大促首屏 |
| 视频商品 | 9:16(竖屏)/ 16:9(横屏)| 直播 / 视频流 |

### 商品图规则
- **背景纯白 / 纯灰**:`#ffffff` 或 `#f5f5f5`,提升商品识别
- **主体居中,留白 8%**:商品占图 92%,边缘 4% 留白
- **不允许背景叠加文字**(除促销标签)
- **去阴影 / 保留真实感**:商品自然光照,不过度修图
- **最小分辨率**:800×800(列表)/ 2000×2000(PDP 主图)

### 角标 / 标识
| 位置 | 元素 | 数量 |
|---|---|---|
| 左上 | 主标签(限时 / 自营 / 直播)| ≤ 1 |
| 右上 | 收藏 / 关闭按钮 | ≤ 1 |
| 左下 | 价格水印(部分卡型)| 0-1 |
| 右下 | 加购按钮(部分卡型)| 0-1 |

**铁律**:**一张商品图最多 3 个角标**。超过则视觉混乱。

## 2. Banner

### 比例
| 场景 | 比例 |
|---|---|
| 首页主 Banner | 16:9 |
| 频道 Banner | 21:9 |
| 楼层卡 Banner | 4:3 |
| 大促首屏 | 全屏(9:16) |

### Banner 规则
- 文字区占 Banner 30-40%,商品/视觉占 60-70%
- 文字与背景对比度 ≥ 4.5:1(防弱视用户看不清)
- CTA 按钮位置一致(右下 / 居中底)
- 不允许 Banner 内嵌真实"虚假倒计时"

## 3. 插画

| 场景 | 风格 | 用途 |
|---|---|---|
| 空状态 | 京东品牌插画风(扁平 + Joy)| 购物车空 / 搜索无结果 |
| 引导 | 同上 + 教学指引 | 新手引导 / 功能介绍 |
| 营销 | 节庆插画(主题感强)| 大促 / 节庆活动 |
| 错误页 | 友好插画 + Joy | 网络异常 / 404 |

**插画使用边界**:[[brand-expression]] —— 插画不抢戏 + Joy 不滥用。

---

## 4. 吉祥物 Joy

详见 [[../../horizontal/brand/mascot-joy.md]]

**核心规则**:
- Joy 出现在节庆 / 引导 / 空状态 / 客服
- Joy 不出现在严肃信息页(订单异常 / 投诉 / 隐私政策)
- Joy 形态有标准库,不允许业务自创

## 5. 加载占位

| Token | 用途 |
|---|---|
| `imagery.placeholder.product` | 商品图加载中 |
| `imagery.placeholder.avatar` | 头像加载中 |
| `imagery.placeholder.banner` | Banner 加载中 |

**实现**:Skeleton + 京东 logo 灰底,过渡到实际图片用 `motion.duration.fast` 淡入。

---

## 6. 图像格式 / 性能

| 场景 | 推荐格式 |
|---|---|
| 商品图 | WebP / AVIF(降级 JPEG)|
| 图标 | SVG(矢量)|
| Banner | WebP + 多尺寸 srcset |
| 插画(精细)| PNG 或 SVG |
| 视频缩略图 | WebP |

**京东图床**:统一 CDN,自动尺寸适配 + 格式协商。

---

## 7. 反例

| ❌ 反面 | 解释 |
|---|---|
| 商品图带"超值/秒杀"水印 | 严重影响商品识别(应用角标) |
| 一张商品图 5+ 角标 | 视觉混乱 |
| Banner 文字对比度 < 4.5:1 | a11y 失败 |
| Joy 出现在订单异常页 | 用户感到被冒犯 |
| 商品图背景五颜六色 | 跨业务体验割裂 |
