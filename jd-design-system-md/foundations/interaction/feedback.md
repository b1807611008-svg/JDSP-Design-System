# 反馈规范 · Feedback

> Loading / 空状态 / 错误 / 成功 4 种反馈。**反馈的核心是让用户知道"系统在做什么 / 我现在该做什么"**。

---

## 1. Loading 反馈

### 类型
| 类型 | 用途 | 时长边界 |
|---|---|---|
| Skeleton(骨架屏)| 整页加载 | 任意 |
| Spinner(旋转器) | 局部加载 / 按钮 loading | 任意 |
| Progress(进度条) | 已知进度(上传 / 下载) | 任意 |
| Toast loading | 短任务(< 3 秒)| ≤ 3 秒 |

### Skeleton 规则
- 灰底渐变动画(`shimmer`)
- 形状还原真实内容(标题 / 图片 / 卡片)
- 加载完成淡入真实内容

### 何时用什么
```
[ 整页首次加载 ] → Skeleton
[ 局部数据加载(列表加载下一页)] → Spinner
[ 按钮提交中 ] → 按钮内 spinner
[ 上传 / 下载 ] → Progress
[ 已知短任务 ] → Toast loading
```

## 2. 空状态

每个空状态包含 4 元素:

```
┌─────────────────┐
│   [插画/图标]    │
│                 │
│   主文案         │
│   辅助说明       │
│                 │
│   [操作按钮]     │
└─────────────────┘
```

### 类型
| 类型 | 主文案示例 | 操作 |
|---|---|---|
| 暂无内容 | "购物车里啥都没有" | "去逛逛" |
| 搜索无结果 | "没找到相关商品" | "换个搜索词" |
| 网络异常 | "网络开小差了" | "重新加载" |
| 未登录 | "登录后查看" | "立即登录" |
| 权限拒绝 | "需要相机权限" | "去设置" |

### 视觉
- 插画/图标尺寸:`icon.size.xxl` (48pt+) 或更大
- 主文案:`body.l` semibold
- 辅助说明:`body.m` `text.secondary`
- 操作按钮:`Button type=primary size=M`

## 3. 错误状态

| 错误级别 | 表达 |
|---|---|
| 字段错误 | Input 下方 helper(本字段)|
| 表单错误 | 顶部 banner |
| 业务错误 | Dialog / Toast / 错误页 |
| 系统错误 | 错误页(刷新 / 联系客服)|
| 网络错误 | 错误页(重试)|

### 错误页结构
同空状态(4 元素),但插画偏向"问题感"(Joy 客服 / 错误图标)。

## 4. 成功反馈

| 场景 | 反馈方式 |
|---|---|
| 加购 / 收藏 | Toast "已加入购物车" + 加购飞动画 |
| 提交订单成功 | 跳转订单详情页 |
| 支付成功 | 全屏成功页 + 倒计时跳转 |
| 评价提交 | Toast "感谢评价" |
| 表单保存 | Toast "已保存" |
| 删除 | Toast "已删除,撤销" |

**核心原则**:
- 短反馈用 Toast(2-3 秒)
- 长反馈跳转(订单 / 支付)
- 成功反馈不阻塞用户(主流程不打断)

## 5. 加购飞动画(京东特色)

PDP / 列表页加购时,商品图飞向购物车 Tab:

| 阶段 | 时长 | 动效 |
|---|---|---|
| 商品图缩小 | 0-100ms | scale 1 → 0.3 |
| 飞行 | 100-400ms | curve path |
| 落入购物车 | 400-500ms | Tab Bar 购物车角标 +1 + bounce |

**实现**:`motion.add-to-cart` 复合动效。

**减少动效模式**:跳过飞行动画,仅 Toast"已加入购物车"+ 角标更新。

---

## 6. 系统空闲检测

长时间无响应:
| 时长 | 反馈 |
|---|---|
| < 1 秒 | 无反馈(瞬时)|
| 1-3 秒 | Spinner |
| 3-10 秒 | Skeleton + 提示文字 |
| 10+ 秒 | 错误页(超时)|

**铁律**:**任何 > 1 秒的等待必须有反馈**,否则用户感觉系统死了。

---

## 7. 反例

| ❌ 反面 | 解释 |
|---|---|
| 加载无反馈 > 1 秒 | 用户感觉死机 |
| 空状态无操作按钮 | 用户不知道下一步 |
| 错误提示笼统("出错了")| 用户不知道怎么办 |
| 成功反馈阻塞主流程 | 打扰用户 |
| Skeleton 形状与实际内容差很大 | 视觉跳动 |
| 加购飞动画无视减少动效设置 | 违反 a11y |
