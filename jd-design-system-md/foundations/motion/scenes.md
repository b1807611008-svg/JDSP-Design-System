# 典型动效场景 · Scenes

> 京东 APP 高频动效场景的标准实现。**这些是品牌资产,不允许各业务自己重做**。

---

## 1. 加购飞动画

### 流程
```
1. 用户点击"加购"按钮
2. 商品图缩小(scale 1 → 0.3)+ 离开原位
3. 沿贝塞尔曲线飞向 Tab Bar 购物车 icon
4. 落入购物车 icon + bounce 反馈
5. 购物车角标 +1
6. Toast"已加入购物车"(可选)
```

### 时长
- 总时长:500ms
- 缩小阶段:0-100ms
- 飞行阶段:100-400ms(主体)
- bounce 反馈:400-500ms

### 缓动
- 缩小:`accelerate`
- 飞行:贝塞尔(自定义路径)
- bounce:`spring`

### 减少动效模式
跳过飞行 → 仅 Toast + 购物车角标 +1

## 2. 大促礼花 / 烟花

### 触发场景
- 618 / 双 11 首屏入场
- 抢购成功
- 优惠券领取(部分场景)

### 实现
- 礼花:Lottie 动画(预设 JSON)
- 时长:`slower` (800ms)
- 自动消失,不阻塞用户

### 减少动效模式
不显示

---

## 3. 翻牌 / 抽奖

### 流程
```
1. 翻牌动画:卡片 Y 轴翻转 180°(0.6 秒)
2. 中奖反馈:卡片 bounce(spring 0.5 秒)
3. 显示奖品 + 领取按钮
```

### 时长
- 翻转:600ms
- bounce:500ms
- 总时长:1.1 秒

---

## 4. 滚动惯性

### 列表滚动
- iOS 系统惯性(默认)
- Android 系统惯性(默认)
- 不允许覆盖系统滚动

### 锚点跳转
- 滚动到锚点:`emphasized` (500ms)
- 锚点 Tab 跟随:同步选中

---

## 5. 转场动画(标准)

| 场景 | 动效 | 时长 |
|---|---|---|
| 列表 → PDP | 共享元素(商品图)| `slow` (500ms) |
| 普通页面 → 二级 | 右滑入(iOS)/ 淡入(Android)| `normal` (300ms) |
| 模态 → 全屏 | 从底部升起 | `normal` (300ms) |
| 全屏 → 模态 | 缩小到模态尺寸 | `normal` (300ms) |
| 返回 | 反向 | 同进入 |

---

## 6. Tab 切换

### 底部 Tab
- 图标:线 → 面 切换 + 京东红着色 + 微 bounce
- 总时长:`fast` (200ms)
- 触感:Light haptic

### 顶部 Tab
- 选中条:跟随 Tab 移动(`emphasized` 200ms)
- 内容区:左右滑动 + 淡入
- 内容区时长:`fast` (200ms)

---

## 7. 下拉刷新

### 阶段
| 阶段 | 视觉 |
|---|---|
| 拖拽中(0-64pt) | 加载 icon 透明度 0 → 1 |
| 释放后 | 加载 icon 旋转(linear 1 圈/秒) |
| 完成 | "已是最新"Toast + 列表回弹 |

### 自定义刷新动画(节庆)
- 春节:Joy 拜年动画
- 618:京东红礼花
- 实现:替换默认刷新 icon 为 Lottie

---

## 8. SKU 选择 Sheet

| 阶段 | 动效 |
|---|---|
| Sheet 升起 | 从底部滑入 + 蒙层淡入 (300ms decelerate) |
| 选规格 | 高亮 + 触感 (100ms) |
| 关闭 | 滑下 + 蒙层淡出 (200ms accelerate) |

---

## 9. 反例

| ❌ 反面 | 解释 |
|---|---|
| 各业务自定义"加购飞动画" | 品牌一致性破坏 |
| 加购飞动画 > 800ms | 用户等待 |
| 大促礼花在主流程出现 | 喧宾夺主 |
| 翻牌动画卡顿(< 60 FPS)| 影响体验 |
| 不响应减少动效 | 违反 a11y |
