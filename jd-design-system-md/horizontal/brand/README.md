## 品牌使用核心原则(对应 [[../../knowledge/philosophy/brand-expression.md]])

1. **京东红是品牌护城河**——主 CTA / 价格 / 品牌区严格使用
2. **Joy 是情感锚点**——节庆 / 引导 / 空状态用,严肃信息不用
3. **大促节庆基因要传承**——618 礼花 / 双 11 渐变 / 春节金色
4. **品牌克制即品牌强大**——非品牌区不滥用红色,反而强化品牌区识别
