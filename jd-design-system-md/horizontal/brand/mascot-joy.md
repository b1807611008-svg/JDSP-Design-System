# Joy 吉祥物使用规范

> 京东 Joy(狗狗)是 6 亿用户的情感锚点。**Joy 是品牌资产,不是免费贴纸**——用得好建立情感连接,用得糟轻浮 / 冒犯。

---

## 1. Joy 形象库

官方形象库(不允许业务自创):

| 形象类型 | 用途 |
|---|---|
| Joy 标准像 | 默认 / 通用 |
| Joy 客服 | 客服 / 帮助中心 |
| Joy 节庆系列(春节 / 中秋 / 圣诞) | 节庆引导 |
| Joy 表情包 | 互动 / 评论 |
| Joy 子品牌系列(医生 Joy / 金融 Joy) | 子品牌差异化 |
| Joy 大促系列(618 / 1111) | 大促引导 |

形象使用必须从官方库引用,不允许 detach 修改。

---

## 2. 应该出现的场景

| 场景 | 示例 |
|---|---|
| 空状态 | "购物车里啥都没有,Joy 邀你逛逛" |
| 节庆引导 | 春节 / 618 / 双 11 主题视觉 |
| 营销 | 限时秒杀 / 大促预告 |
| 客服 / 帮助 | Joy 客服形象 |
| 引导教学 | 新功能介绍 / 使用引导 |
| 评论 / 互动 | Joy 表情包 |

---

## 3. 不应出现的场景

| ❌ 场景 | 解释 |
|---|---|
| PDP / 购物车 / 结算 / 支付 | 主流程不能被打扰 |
| 订单详情 / 售后 / 投诉 | 严肃信息 |
| 隐私政策 / 协议 / 法律页 | 严肃内容 |
| 错误反馈(网络异常 / 系统错误) | 用户已受挫,Joy 显得轻浮 |
| 财务相关(白条 / 提现 / 退款失败) | 严肃 |

---

## 4. Joy 尺寸

| 场景 | 尺寸 |
|---|---|
| 空状态(主插画) | ≥ 120pt |
| 引导浮层 | 80pt |
| 表情包 | 32-48pt |
| 客服头像 | 40pt |
| 角标位(节庆) | 24pt |

---

## 5. Joy 颜色

| 背景 | Joy 颜色 |
|---|---|
| 浅色 | 标准全色 Joy |
| 京东红底 | 白色描边 Joy 或反白 |
| 深色 | 标准全色 Joy(浅色版本) |

---

## 6. Joy 与节庆基因

每个节庆 Joy 有专属形象:
- 春节 Joy:戴红帽 + 福字
- 618 Joy:站在礼花上
- 双 11 Joy:举优惠券
- 中秋 Joy:吃月饼

形象由品牌设计组维护,业务方不允许私自创作。

---

## 7. 反例

| ❌ 反面 | 解释 |
|---|---|
| 业务方自创 Joy 形象 | 破坏品牌一致性 |
| Joy 出现在订单异常页 | 用户感觉被冒犯 |
| Joy 尺寸过大遮挡核心信息 | 喧宾夺主 |
| Joy 用错节庆形象(春节用 618 Joy) | 视觉违和 |
| 主流程页面贴 Joy(PDP / Cart) | 影响转化 |
