# 装饰图形 · Decoration

> 大促 / 节庆 / 营销场景使用的装饰图形。**装饰要为内容服务,不抢戏**。

---

## 装饰风格

| 风格 | 用途 |
|---|---|
| **扁平**(flat) | 默认,日常营销 |
| **渐变**(gradient) | 大促 / 节庆 |
| **3D**(立体) | 顶级大促首屏 / 节庆主视觉 |
| **拟物**(skeuomorphic) | 不主用(京东视觉传统不符) |

---

## 装饰元素清单

| 元素 | 用途 |
|---|---|
| 礼花 / 烟花 | 抢购成功 / 大促首屏 |
| 倒计时背景 | 限时秒杀 |
| 优惠券装饰 | 优惠券领取页 |
| 节庆元素(灯笼 / 月饼 / 玫瑰) | 对应节庆 |
| 渐变光晕 | 大促头图 |
| 闪光 / 星星 | 营销亮点 |

---

## 装饰使用边界

✅ **允许**:
- 大促首屏(顶部 1-2 屏)
- 频道页 Banner
- 节庆主题区
- 营销楼层

❌ **不允许**:
- PDP 主图区
- 购物车 / 结算 / 支付主流程
- 订单 / 售后 / 客服页
- 信息密度高的列表流(双列卡)

---

## 装饰密度

| 区域 | 装饰占比 |
|---|---|
| 大促首屏 | 可达 40-60% |
| 频道页 | 20-30% |
| 主流程页 | ≤ 10%(个别图标元素)|
| 信息流 | ≤ 5% |

---

## 实现方式

| 元素 | 推荐实现 |
|---|---|
| 静态装饰 | SVG / WebP |
| 动态装饰 | Lottie(JSON) |
| 复杂动效 | MP4 / 透明背景视频(谨慎使用)|

**性能铁律**:
- 单帧渲染 < 16ms(60 FPS)
- 移动端 Lottie 文件 < 200KB
- 装饰元素总内存占用 < 5MB

---

## 反例

| ❌ 反面 | 解释 |
|---|---|
| 装饰挤占商品识别 | 影响转化 |
| 装饰动效不响应减少动效 | 违反 a11y |
| 装饰文件过大(> 1MB) | 加载慢 |
| 装饰风格与节庆不符 | 视觉违和 |
| 装饰盖到主 CTA | 阻碍主操作 |
