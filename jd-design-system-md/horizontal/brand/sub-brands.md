# 子品牌差异化边界

> 京东超市 / 京东健康 / 京东金融 / 京东物流 / 京东工业 等子品牌的视觉差异化规则。**保留京东标识 + 允许子品牌特色**。

---

## 1. 子品牌矩阵

| 子品牌 | 简称 | 主业务 | 辅助色 |
|---|---|---|---|
| 京东商城(主)| Retail | 综合电商 | - |
| 京东超市 | JD Supermarket | 食品 / 日用 | 鲜橙 `#ff6b35` |
| 京东生鲜 | JD Fresh | 生鲜 / 30 分钟达 | 草绿 `#00a870` |
| 京东健康 | JD Health | 医药 / 健康 | 医蓝 `#1677ff` |
| 京东金融 | JD Finance | 白条 / 金条 / 理财 | 黑金 `#000` + `#f5a623` |
| 京东物流 | JD Logistics | 快递 / 配送 | 物流蓝 `#0066cc` |
| 京东工业 | JD Industrial | B 端工业品 | 钢蓝 `#3a4754` |
| 京东国际 | JD International | 跨境电商 | 主京东红 |

---

## 2. 必须保留的京东标识

| 元素 | 出现位置 |
|---|---|
| 京东 Logo | 启动屏 / 顶部导航 / 我的页 |
| 京东红 | 主 CTA / 价格 |
| 京东品牌字体 | 品牌区文字 |
| 节庆 Joy | 大促 / 节庆引导(可子品牌变体)|

**铁律**:**所有子品牌必须保留京东 Logo + 京东红主 CTA**。

---

## 3. 可以差异化的元素

| 元素 | 差异化方式 |
|---|---|
| 辅助品牌色 | 子品牌特色色(如京东健康的医蓝) |
| 频道入口图标 | 子品牌专属风格 |
| Joy 形象变体 | 京东健康的 Joy 医生 |
| 大促主题 | 子品牌专属节庆 |
| 装饰图形风格 | 与业务调性匹配 |
| 字体辅助使用 | 京东金融可用更严肃字体 |

---

## 4. 各子品牌设计调性

### 京东超市
- 调性:亲民 / 实惠 / 快速
- 辅助色:鲜橙
- 视觉:商品图密集 + 价格突出

### 京东生鲜
- 调性:新鲜 / 速达 / 信任
- 辅助色:草绿
- 视觉:产地标识 + 30 分钟达 + 保鲜标记

### 京东健康
- 调性:专业 / 严谨 / 安心
- 辅助色:医蓝
- 视觉:医生认证 + 处方标识 + 隐私保护
- Joy 变体:Joy 医生

### 京东金融
- 调性:严谨 / 高端 / 安全
- 辅助色:黑金
- 视觉:克制 + 金融数据展示
- **特殊**:不主用 Joy(金融严肃感)

### 京东物流
- 调性:专业 / 高效 / 透明
- 辅助色:物流蓝
- 视觉:配送时效 + 物流轨迹 + 司机信息

---

## 5. 子品牌的"父品牌"承诺

每个子品牌进入主品牌:
- 顶部导航有"京东 / 子品牌"切换入口
- 用户可在主站 / 子品牌之间切换
- 优惠券 / 京豆 / 会员体系**部分共通**

---

## 6. 反例

| ❌ 反面 | 解释 |
|---|---|
| 子品牌完全脱离京东红 | 用户认知断裂 |
| 子品牌主 CTA 用辅助色 | 体验割裂 |
| 子品牌私自创 Joy 形象 | 破坏品牌资产 |
| 子品牌 Logo 替换"京东" | 法律 / 商业问题 |
| 子品牌大促与主站完全错峰 | 资源浪费 |
