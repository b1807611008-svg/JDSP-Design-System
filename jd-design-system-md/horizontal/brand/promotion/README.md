## 统一基因

所有大促共享:
- **京东红基底**:主 CTA / 价格仍然京东红(铁律)
- **Joy 节庆形象**:每个节庆有专属 Joy
- **倒计时元素**:统一倒计时组件 [FlashSaleTimer]
- **优惠条结构**:满减 / 折扣 / 优惠券统一展示规则
- **大促按钮风格**:胶囊型(`radius.full`)+ 渐变(主题包覆盖)

---

## 大促主题切换

```
Default Theme   → 日常主题(纯色京东红)
↓ 大促前 7 天预热
Promotion Theme → 大促主题(渐变 / 胶囊 / 节庆装饰)
↓ 大促结束
Default Theme
```

切换通过 Token 主题层完成,**组件代码不感知**。详见 [[../../../ai-mechanism/token-sync.md]]

## 大促首屏 vs 主流程

| 区域 | 视觉策略 |
|---|---|
| 大促首屏(顶部 1-2 屏) | 品牌氛围优先(礼花 / 渐变 / 装饰) |
| PDP / 购物车 / 结算 / 支付 | 聚焦实效优先(克制装饰)|

---

## 大促资产管理流程

1. 大促前 8-12 周:品牌设计组开始准备资产包
2. 大促前 4 周:资产包评审 → 进入主题包
3. 大促前 2 周:CDN 下发 → 客户端预加载
4. 大促前 7 天:开始切换主题(预热)
5. 大促当天:主题完全启用
6. 大促结束 3 天后:切回 Default
