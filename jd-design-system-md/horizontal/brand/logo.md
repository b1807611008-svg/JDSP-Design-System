# Logo 使用规范

> 京东 Logo 是品牌最浓缩的视觉表达。**安全距离 / 最小尺寸 / 禁用变形** 是三大铁律。

---

## 1. Logo 类型

| Logo | 用途 |
|---|---|
| 京东中文标 | 中文场景主用 |
| JD.COM 英文标 | 国际版 / 英文场景 |
| 京东 + 狗狗 组合标 | 启动屏 / 大促首屏 |
| 简化标(小尺寸) | favicon / 角标位 |

---

## 2. 安全距离

Logo 周围必须保留安全距离 = Logo 高度的 50%(最小)

```
┌─────────────────┐
│      ⬆ 50%      │
│  ←50%─[Logo]─→ │
│      ⬇ 50%      │
└─────────────────┘
```

**禁止**:文字 / 图形 / 其他 Logo 进入安全距离。

---

## 3. 最小尺寸

| 场景 | 最小尺寸 |
|---|---|
| 启动屏 | 高度 ≥ 80pt |
| 顶部导航 | 高度 ≥ 24pt |
| 角标位 / favicon | ≥ 16pt(用简化标) |

---

## 4. Logo 颜色

| 背景 | Logo 颜色 |
|---|---|
| 白底 | 京东红 `color.brand.primary` |
| 浅色背景 | 京东红 |
| 京东红底 | 白色 |
| 深色 | 白色 |
| 渐变背景 | 白色或带白色描边 |

---

## 5. 禁用变形

❌ **禁止**:
- 拉伸 / 压扁
- 旋转
- 加阴影 / 描边(默认无阴影)
- 替换字体
- 任意改色(除黑/白/京东红组合)
- 过度切割 / 残缺

---

## 6. Logo 在子品牌中

| 子品牌 | Logo 处理 |
|---|---|
| 京东超市 | "京东" + "超市"组合,京东在前 |
| 京东健康 | "京东" + "健康"组合 |
| 京东金融 | "京东" + "金融"组合 |
| 京东国际 | "京东" + "国际" |

**铁律**:**所有子品牌必须有"京东"标识**,不允许完全脱离京东 Logo。

详见 [[sub-brands.md]]

---

## 7. 反例

| ❌ 反面 | 解释 |
|---|---|
| Logo 周围紧贴文字(无安全距离) | 视觉拥挤 |
| Logo 拉伸变形 | 品牌资产损坏 |
| Logo 颜色随意改 | 品牌识别度下降 |
| 子品牌完全脱离"京东"标识 | 用户认知断裂 |
