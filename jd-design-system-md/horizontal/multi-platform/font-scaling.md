# 字号缩放 · Font Scaling

> 系统字号 85%-150%,**所有页面必须在 150% 字号下不崩**。

---

## 1. 系统字号支持

iOS:Settings → Display & Brightness → Text Size(7 阶,85%-180%)
Android:Settings → Display → Font Size(4 阶,85%-130%,Android 12+ 支持更大)

京东 APP 支持:
| 字号 | 占比 | 用途 |
|---|---|---|
| 85% | < 5% 用户 | 信息密度高的用户 |
| 100% | 默认 | 80%+ 用户 |
| 115% | 中老年用户 | 10-15% 用户 |
| 130% | 适老模式默认 | 老人用户 |
| 150% | 极端字号 | 弱视用户 |

---

## 2. 实现机制

字号 Token 引用系统字号缩放:
```css
font-size: calc(15pt * var(--font-scale));
```

`--font-scale` 由系统注入,1.0 = 100%。

---

## 3. 布局适配

### 单行文字
- 小尺寸:可截断(省略号)
- 关键文字:换行 / 缩字号
- 价格:**永远不允许截断**

### 按钮
- 文字超长:换行(2 行内)+ 按钮高度自适应
- 极端情况:缩字号(降一阶)

### 卡片
- 卡片高度自适应文字
- 不允许内容溢出

---

## 4. 测试矩阵

每个页面上线前必跑:
- [ ] 85% 字号:布局紧凑但完整
- [ ] 100% 字号:默认
- [ ] 130% 字号:适老模式
- [ ] 150% 字号:极端,布局不崩

---

## 5. 适老模式自动切换

用户在系统设置 ≥ 130% 时,京东 APP 自动建议进入适老模式:
- 单列流(不双列)
- CTA 加大 + 描边强化
- 装饰简化(去除部分动效)

详见 [[../../knowledge/philosophy/inclusive.md]]

---

## 6. 多语言考虑

不同语言字符宽度不同:
- 中文 1 字 ≈ 英文 2 字符
- 西语字符更宽
- 阿拉伯语 RTL 布局(京东全球版)

布局必须在 150% + 西语 / 阿语下不崩。

---

## 7. 反例

| ❌ 反面 | 解释 |
|---|---|
| 字号硬编码不响应系统设置 | 老人用户体验差 |
| 150% 字号下按钮文字截断 | 老人看不全 |
| 价格被截断 | 影响核心识别 |
| 布局在 150% 下溢出屏幕 | 内容看不到 |
| 不支持适老模式自动切换 | 用户每次手动调 |
